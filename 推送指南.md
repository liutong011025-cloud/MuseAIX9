# 推送到 GitHub 仓库 MuseAIX9

## 快速推送步骤

### 方法 1: 使用 PowerShell 脚本（推荐）

1. 在项目根目录打开 PowerShell
2. 运行以下命令：

```powershell
.\push-to-github.ps1
```

### 方法 2: 手动 Git 命令

如果脚本无法运行，可以手动执行以下命令：

```powershell
# 1. 检查 Git 是否已安装
git --version

# 2. 如果未初始化，初始化仓库
git init
git branch -M main

# 3. 添加远程仓库
git remote add origin https://github.com/liutong011025-cloud/MuseAIX9.git
# 如果已存在，更新远程地址：
git remote set-url origin https://github.com/liutong011025-cloud/MuseAIX9.git

# 4. 添加所有文件
git add .

# 5. 提交更改
git commit -m "Update: Fix Prisma build issue and Dify API configuration for Vercel deployment"

# 6. 推送到 GitHub
git push -u origin main
```

### 如果推送失败

如果遇到以下情况：

1. **需要身份验证**：
   - 使用 GitHub Personal Access Token (PAT) 作为密码
   - 用户名：你的 GitHub 用户名
   - 密码：你的 Personal Access Token

2. **远程仓库已有内容**：
```powershell
git pull origin main --allow-unrelated-histories
git push -u origin main
```

3. **强制推送（谨慎使用）**：
```powershell
git push -u origin main --force
```
